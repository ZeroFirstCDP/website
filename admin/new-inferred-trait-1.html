Step 1: Select the type of trait to create

Each type of computed trait serves a different purpose. Choose the one that best fits your needs.

Event Counter

An Event Counter trait stores a count of an event over a period of time. You can also use event properties to only specific types of events.

Most Frequent

A most frequent user-level computed trait will return the most common value for an event property. In the case of a tie, the first alphabetical value is returned.
